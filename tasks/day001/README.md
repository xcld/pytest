### 3.if条件语句句的基本结构？
三种结构：
1. 条件动作
```python
if boolean_expression1:
    suite1
```
2. 分支型
```python
if boolean_expression1:
    suite1
else
    else_suite
```
3. 多分支
```python
if boolean_expression1:
    suite1
elif boolean_expression2:
    suite2
elif boolean_expressionN:
    suiteN
else:
    else_suite
```
